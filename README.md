# GGScript Compiler

## Prerequisites
- Python 3.8+
- pip
- Virtualenv (recommended)

### Installation 

1. **Clone the repository**
```
$ git clone https://github.com/wkwk08/GGScript.git
$ cd GGScript
```

2. **Create and activate a virtual environment**
```
$ python -m venv venv

# Windows
$ venv\Scripts\activate

# macOS/Linux
$ source venv/bin/activate
```

### Usage 

#### Run 

Launch the native desktop Tkinter GUI:
```
$ python main.py
```
